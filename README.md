# 🎰 Lottery Ticket - Solidity

A simple decentralized lottery smart contract built on Ethereum using Solidity.

This project allows users to participate in a lottery by sending **1 ETH** to the contract. Once the required number of participants join, the manager can initiate a draw, and the contract automatically transfers the entire prize pool to a randomly selected winner.

## 🚀 Features

* Secure participant registration
* Fixed entry fee (1 ETH)
* Manager-controlled winner selection
* Automatic prize distribution
* Lottery reset after every round
* Fully on-chain execution

## 🛠 Tech Stack

* Solidity ^0.8.19
* Ethereum Blockchain
* Remix IDE
